## 첫프로젝트를 하면서 느낀점

일단 it분야가 처음이었기에 용어나 프로그램을 사용하는것 자체가 굉장히 생소하게 느껴졌습니다
이걸 왜 써야 하나, 여기에 왜 이런글자가 쓰이나 궁금한 것도 많았지만
교과서를 두고 수업을 하는게 아니라
강사의 커리큘럼대로 진도를 나가야했기에 항상 진도를 따라가기에 급급했던것 같습니다
그래서 나중에 시간이 남을때마다 틈틈이 이지스퍼블리싱의 Doit html, css책을 보면서 개념과 반복학습을 해왔습니다.
시간이 많이 필요로 하긴했지만, 프로그램을 사용하고 기초를 잡는데는 많은 도움이 되었던것 같습니다.
